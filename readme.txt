      間引きフレーム補完 フィルタプラグイン ver0.01  by MakKi
-----------------------------------------------------------------------

【機能】

　  fpsを変更することで間引かれるフレームを
  次、または前のフレームに任意の不透明度で重ねます


【インストール】

　delcomp.auf を AviUtl の認識可能なフォルダに入れてください


【使用方法】

  ・強度 スライダ
	重ねる画像の不透明度を変更します

  ・前フレームに重ねる チェック
	間引かれるフレームを前のフレームに重ねます

  ・後フレームに重ねる チェック
	間引かれるフレームを後ろのフレームに重ねます

  ・キーフレームには重ねない
	キーフレームに設定されているフレームには重ねません
